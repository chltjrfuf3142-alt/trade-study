## 💻 컴퓨터에 앱으로 설치하기

### Chrome 브라우저
1. 위 주소를 **Chrome**으로 접속
2. 주소창 오른쪽에 나타나는 **⬇️ 설치 아이콘** 클릭
   - 또는 오른쪽 상단 `⋮` → **"앱 설치"** 클릭
3. **"설치"** 버튼 클릭
4. ✅ 바탕화면에 아이콘 생성 → 독립 창으로 실행됨!

### Edge 브라우저
1. 위 주소를 **Edge**로 접속
2. 주소창 오른쪽 **"앱 사용 가능"** 아이콘 클릭
3. **"설치"** → 바탕화면 + 시작메뉴에 추가됨

### 삭제 방법
- 앱 창 상단 `⋮` → **"앱 제거"**
- 또는 Windows 설정 → 앱 → 해당 앱 제거

## 📱 iPhone(iOS)에 앱으로 설치하기

1. **Safari**로 위 주소 접속 (⚠️ 반드시 Safari 사용!)
2. 하단의 **공유 버튼** (⬆️) 탭
3. 목록에서 **"홈 화면에 추가"** 선택
4. 이름 확인 후 **"추가"** 탭
5. ✅ 홈 화면에 앱 아이콘 생성 → 전체화면으로 실행됨!

> 💡 Chrome이나 다른 브라우저에서는 "홈 화면에 추가" 기능이 없습니다. 반드시 **Safari**를 사용하세요.

## 📱 Android에 앱으로 설치하기

1. **Chrome**으로 위 주소 접속
2. 화면 하단에 **"홈 화면에 추가"** 배너가 나타남
   - 또는 오른쪽 상단 `⋮` → **"앱 설치"** 클릭
3. **"설치"** 탭
4. ✅ 홈 화면에 앱 아이콘 생성!

## 🔐 Google 계정 연동 (클라우드 동기화)

### 연동 방법
1. 앱 홈 화면에서 **"🔐 Google 로그인"** 버튼 클릭
2. Google 계정으로 로그인
3. 프로필 사진과 **"☁️ 동기화됨"** 메시지가 표시되면 완료!

### 연동하면 좋은 점

| 기능 | 로그아웃 상태 | 로그인 상태 |
|------|:---:|:---:|
| 오답노트 저장 | 해당 기기만 | ☁️ 모든 기기 |
| 학습 통계 | 해당 기기만 | ☁️ 모든 기기 |
| 기기 변경 시 | ❌ 데이터 유실 | ✅ 자동 복원 |
| 폰↔PC 동기화 | ❌ 불가 | ✅ 실시간 동기화 |

> 💡 예를 들어 **폰에서 퀴즈를 풀고 틀린 문제**가 있으면, **PC에서 로그인해도 같은 오답노트**가 보입니다!

### 주의사항
- 로그인하지 않아도 앱은 정상적으로 사용 가능합니다 (데이터가 해당 기기에만 저장됨)
- 여러 기기에서 동시에 사용 가능합니다
- 로그아웃하면 클라우드 동기화가 중지되지만, 이미 저장된 데이터는 유지됩니다
